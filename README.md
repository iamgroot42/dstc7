# dstc7
DSTC7 challenge

Basic codebase from : https://github.com/IBM/dstc7-noesis

## Additional features

* BiDirectional LSTM
* Sum, Mean and Max-pool based feature encoding
* CNN based max-pooling of features
* Attention
* Fast GRNN
* Using C-DSSM based sentence (pass on in input data) embeddings
